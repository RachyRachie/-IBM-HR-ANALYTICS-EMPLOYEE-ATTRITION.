# Employee Attrition Analysis

## Project Overview
This project analyzes employee attrition using HR data to identify the factors that contribute to employees leaving an organization. The goal is to uncover trends and provide data-driven insights that can help improve employee retention and workplace satisfaction.

## Dataset
The dataset contains employee demographic, job, and performance information, including:
- Age
- Department
- Job Role
- Monthly Income
- Business Travel
- Overtime
- Work-Life Balance
- Job Satisfaction
- Years at Company
- Attrition (Target Variable)
- And other HR-related features.

## Objectives
- Analyze employee attrition patterns.
- Identify the key factors influencing employee turnover.
- Explore relationships between employee characteristics and attrition.
- Generate actionable insights through data analysis and visualization.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Correlation analysis
- Employee attrition insights and recommendations

## Project Outcome
The analysis highlights the major drivers of employee attrition, such as overtime, job satisfaction, income level, work-life balance, and years at the company. These findings can support HR teams in making informed decisions to improve employee engagement and reduce turnover.
